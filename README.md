# Look Out

In charge of the observation of the code for hazards.

## Installation

Add this line to your application's Gemfile:

```ruby

gem 'look-out'

```

And then execute:

```bash

$ bundle

```

```ruby

LookOut.configure do |config|
  config.api_key = 'your-api-key'
end

```
